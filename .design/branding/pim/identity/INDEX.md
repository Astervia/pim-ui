## Summary

**Direction:** Terminal verde — phosphor-screen green as the dominant color language. Dark green-tinted ground (#080e09), pale phosphor text (#a8d5a2), electric signal green (#22c55e), amber accent (#e8a84a) for warnings. Monospaced throughout.

**Logo:** `█ pim` — cursor-block + wordmark. The cursor is the logo. Icon-only: `█` in signal green.

**Color:** Monochromatic green family. Signal green (#22c55e) for active/connected/success. Amber (#e8a84a) for warnings. No blue, no purple, no gradients.

**Type:** Geist Mono (headings, labels, wordmark) + Geist (prose) + JetBrains Mono (code/CLI output). No proportional type for UI elements.

**Imagery:** CLI output panels, ASCII topology diagrams, protocol field maps. No photography.

**Applications:** Documentation hero, GitHub README, CLI help, error messages, route status — every surface designed with the same discipline as the protocol spec.
